The Movie Picker Webapp

Premised: This webapp will allow to help users decide on 
what movie they want to watch.

Features:
p0
-The user can create an account 
-They will be logged in on session
-They can chose what movie they want by swiping
p1
-The app will have a sort of room feature where multiple users can be in the same room
-The movies will be hardcoded into the database
p2
-The app will scrape movies from movie listing website(s)
